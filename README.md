#  Poetic Chatbot

A conversational AI that answers every user message with original, short poetic verses – using Google's Gemini 2.0 Flash model.

##  Description
This chatbot transforms ordinary questions into vivid, emotional poetry. Instead of “Washington D.C.”, it might reply:

> *Where marble domes kiss the sky,  
> And the eagle's cry whispers history's sigh.*

The project demonstrates **prompt engineering** to steer a large language model into a consistent creative persona.

##  Features
- **Poetic responses** – 2–4 lines of verse, always in rhyme or free verse.
- **Streaming text** – replies appear word‑by‑word for a “typing” effect.
- **Built with Gemini** – uses `google-genai` and the `gemini-2.0-flash` model.
- **Google Colab ready** – runs entirely in your browser.
